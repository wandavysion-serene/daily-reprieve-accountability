# SAA Meeting Flow App

A structured meeting script and presentation tool built for a specific Sex Addicts Anonymous (SAA) meeting.

This application provides a guided, step-by-step meeting flow including:

- Structured script presentation
- Paragraph and list formatting
- Image support
- Timed meditation segments
- Conditional branching (e.g., newcomer script)
- Serenity Prayer toggle (I / We versions)
- Meeting navigation controls
- Host meta-instructions

The app functions as a live meeting companion for the host, ensuring consistency, pacing, and clarity throughout the meeting.

---

## Purpose

This application was created to:

- Support structured and consistent meeting facilitation
- Reduce cognitive load for the meeting host
- Provide timed segments for meditation and readings
- Centralize meeting materials into a clean, navigable interface

This project is intended for private use within a specific meeting context.
